# Getting started
- Clone the repository
```
git clone  <git lab template url> <project_name>
```
- Install dependencies
```
cd <project_name>
npm install
```
- Build and run the project
```
npx node index.js
```
  Navigate to `http://localhost:3000`
  
  - Category List
  http://localhost:3000/categories

  - Cook List
  http://localhost:3000/cooks
